# 🤖 Niggesh

> An experimental human-based system currently stuck in beta.

## 📖 Description
Niggesh is a highly unpredictable organism that attempts to function like a normal human but frequently encounters runtime errors.

This project has been under development for many years, but major bugs still remain.

---

## ⚙️ System Requirements

- 1 functioning brain (recommended but not detected)
- WiFi connection for memes
- Food supply every few hours
- Patience from everyone nearby

---

## 🚀 Features

| Feature | Status |

## 🐛 Known Bugs

- Says "I'll do it later" and never does
- Opens 15 tabs to study and ends up watching YouTube
- Confidently wrong about everything
- Needs 3–5 business days to reply to messages
