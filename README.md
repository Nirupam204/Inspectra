# Inspectra

A clean, automated website testing project for validating web flows and behavior reliably.

## Overview

Inspectra helps you run repeatable website checks during development and QA.  
It is designed to be simple to set up, easy to run locally, and ready for test automation workflows.

---

## Installation

```bash
git clone https://github.com/Nirupam204/Inspectra.git
cd Inspectra

pip install -r requirements.txt

playwright install chromium

ollama pull phi3
```

## Run Project

```
python main.py
```
